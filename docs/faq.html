Frequently asked questions

Are the apps free?

Yes. Current public releases are free to download and use. Optional support through Buy Me a Coffee helps fund development and testing.

Do the apps collect personal data?

No. Core functionality runs locally and no account is required. TidySweep for Windows uses the internet only after you request an official driver update check.

How do I install TidySweep on Windows?

  1. Download TidySweep-Setup-1.2.1.exe from the official release page.
  2. Open the installer and follow the standard setup steps.
  3. Launch TidySweep from the Start menu.

Why might Windows show a SmartScreen warning?

The website installer currently uses a TidySweep Software self-signed certificate, not a publicly trusted commercial Authenticode certificate. SmartScreen or antivirus software may therefore display a warning.

Download only from the official GitHub release. The Microsoft Store edition is undergoing certification and will use Microsoft's trusted Store distribution process when published.

How do I install a Mac app?

  1. Download the DMG from the official release page.
  2. Open it and drag the app into Applications.
  3. Launch the app from Applications.

Why does macOS show a security warning?

Current direct-download Mac builds are ad-hoc signed and are not yet Apple-notarized.

If macOS blocks the first launch, open System Settings → Privacy & Security, find the blocked-app message, and choose Open Anyway only if you downloaded the app from our official GitHub release page. We plan to add Developer ID signing and notarization.

Why does TidySnipping request Screen Recording access?

macOS requires this permission for applications that capture screen content. TidySnipping processes captures locally and saves PNG files to your Desktop.

Where can I report a problem?

Use our bug-report form or email tidysweepsoftware@gmail.com.
